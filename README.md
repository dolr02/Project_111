Pokud uživatel vybere takové číslo textu, které není v zadání, 
program jej upozorní a skončí, pokud uživatel zadá jiný vstup než číslo, 
program jej rovněž upozorní a skončí. Sumarizace počtu slov,počtu slov začínajících velkým písmenem,
počtu slov psaných velkými písmeny, počtu slov psaných malými písmeny, počtu čísel (ne cifer),
sumu všech čísel (ne cifer) v textu.
